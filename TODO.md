# OpenStorm VR - Developer Todo List

This file tracks upcoming features, enhancements, and tasks for OpenStorm VR.

## Upcoming Features

- [ ] **Update Topo Map**: Refresh the underlying topographic map provider or tile set for better resolution and accuracy.
- [ ] **Lat/Lon GPS Coordinate Placement**: Add the ability to accurately place custom markers, annotations, or spots using real-world Latitude and Longitude coordinates.
- [ ] **VR Camera Placement**: Allow the user's VR camera/pawn to be instantly teleported to specific Lat/Lon markers to view storms from the ground at exact locations.
- [ ] **VR Settings Menu Demystification**: Overhaul the existing settings menu (currently in ImGui) into a simplified, immersive, and VR-friendly UI that is easy to navigate with motion controllers.
